## Choose what you are setting up

Decide which Aware products you need:

- **RestaurantPOS** тАУ in-experience ordering and payments (inside Roblox via `AG | RestaurantPOS`).  
- **TicketPOS** тАУ ticket sales and access control (inside Roblox via `AG | TicketPOS`).  
- **EasyAPI** тАУ Roblox Lua helper for calling Aware HTTP APIs (including Dynaban and Atlas in the future).

You can add products one-by-one. RestaurantPOS and TicketPOS run fully in your Roblox game, while EasyAPI talks to AwareтАЩs backend services.

## Step 1: Create an Aware project (for EasyAPI and backend APIs)

<Steps>
  <Step title="Create your Aware project">
    1. Sign in to your Aware dashboard.  
    2. Create a new project for your experience (for example, **Main Restaurant Tycoon**).  
    3. Note your **Project ID** and **API key** тАУ you will use these with EasyAPI and HTTP APIs (Dynaban, Atlas).
  </Step>

  <Step title="Configure environments">
    We recommend at least two environments:

    - **Sandbox** тАУ for local testing and private Roblox places.
    - **Production** тАУ for your live game.

    For each environment, create a separate API key and restrict usage to the correct servers.
  </Step>
</Steps>

## Step 2: Connect from Roblox (EasyAPI)

If you are integrating backend APIs (like Dynaban) from Roblox, use EasyAPI so you do not have to write raw HTTP requests.

<Steps>
  <Step title="Install EasyAPI Lua module">
    1. Add the EasyAPI ModuleScript to `ReplicatedStorage` or `ServerScriptService`.  
    2. Configure your `ProjectId` and `ApiKey` inside the module or via a config script.  
    3. Make sure only server-side code can access your API key.
  </Step>

  <Step title="Make your first call">

```lua
local EasyAPI = require(game.ServerScriptService.EasyAPI)

local client = EasyAPI.new({
  projectId = "your-project-id",
  apiKey = "your-server-api-key",
  environment = "sandbox",
})

local result, err = client:call("dynaban.evaluate", {
  userId = player.UserId,
  placeId = game.PlaceId,
})

if err then
  warn("Dynaban request failed", err)
  return
end

if result.riskScore >= 70 then
  player:Kick("Your account has been banned by Aware.")
elseif result.riskScore >= 50 then
  player:Kick("Your account has been flagged by Aware.")
end
```

  </Step>
</Steps>
